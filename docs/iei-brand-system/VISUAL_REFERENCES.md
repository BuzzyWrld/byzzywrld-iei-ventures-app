## 🎨 Tab's Taste Manifesto (the 6 constants across ALL her work)

Even though every brand gets its own look, these constants always hold true. The AI's taste rules should enforce these regardless of which aesthetic it picks:

### Constant 1 — Real people, not stock
- Documentary/editorial photography of actual humans the brand serves
- Candid energy preferred over posed
- Black + brown faces centered (across ~80% of Tab's work)
- Photos should look like real life, not Shutterstock smiles

### Constant 2 — Bold type with intent
- Every piece has at least one TYPOGRAPHIC moment that takes up real estate
- Display types are heavy weights (Bebas Neue · Anton · Manrope Black · Inter Black · Impact-family)
- Type carries meaning — it's never decorative filler

### Constant 3 — Concept-driven imagery
- Visuals carry meaning (puzzle pieces for "Without Walls" mental health · film strips for fashion drops · bridges for mentorship · raised fists for movement · butterfly gestures for transformation)
- Symbolism over decoration

### Constant 4 — Color commits
- When a color is picked, it COMMITS — fills space, blocks regions, anchors hierarchy
- Pastels rarely appear; when they do, they're saturated/punched (hot pink not blush)
- Color-blocking is a recurring layout move

### Constant 5 — Information hierarchy is sharp
- Eye flows naturally from biggest type → image → supporting copy → CTA/QR
- Data + stats often treated as design elements (the StopTheStigma 20% / 3x stat cards)
- Negative space used confidently, never accidentally

### Constant 6 — Baltimore + community grounding
- Local institutions, local founders, local energy throughout
- Even non-Baltimore work carries the same "community-rooted" energy
- No corporate-sterile detachment

## 🎯 AI Decision Framework — picking the right aesthetic per customer

When the brand-playbook skill generates assets, it should walk through this decision tree:

```
1. PERSONA + INDUSTRY → narrows the family
   - Wellness / spiritual → warm earth + serif accents
   - Tech / SaaS → modern sans + restrained palette + concept mark
   - Streetwear / fashion → texture + bold type + photography-led
   - Mental health / social impact → bold + activist + real-people-centered
   - Hospitality / nightlife → mood lighting + monogram + sophisticated
   - Community / nonprofit → real-action photography + warm palette + mission-first
   - Consulting / B2B → editorial + confident type + intentional whitespace

2. VIBE ANSWERS (car / animal / anthem / 3-word) → narrows the personality
   - Vintage car answer → restrained palette, more serif moves, less neon
   - Bronco / Jeep answer → utility palette, rugged textures
   - Tesla answer → minimalist modern, future-forward
   - Owl animal → calm, observant, restraint
   - Whale animal → big presence, slow gravitas
   - Sade / Mac Miller anthem → quiet luxury vibe
   - Beyoncé Renaissance anthem → bold, joyful, takes up space

3. ADMIRED BRANDS (Q25) → directional reference (NOT to copy)
   - Mimic STRUCTURE not content
   - Pull color/type/layout principles, not assets

4. UPLOADED VISUAL REFS (Q21) → highest signal
   - If customer uploaded refs, those dominate the direction
   - AI should describe what it sees + reflect it back in generation choices

5. OPPOSITE-OF answer (Q20) → guardrails
   - Whatever customer said NOT to feel like → AI moves OPPOSITE direction
```

**Output of decision: one of the 7 aesthetic families above (or a hybrid), plus specific palette + type + photography direction tuned per customer.**

## 🏷️ BRAND BLUEPRINT'S OWN VISUAL IDENTITY (the product itself)

**This is the look Brand Blueprint as a product wears — NOT what it generates for customers.**

Brand Blueprint inherits from IEI's existing visual identity, which has TWO coexisting modes:

### Mode A — IEI Social Content Vibe (for product surfaces: chat UI, emails, in-app moments, watermarks)

Source: the "Marketing won't fix" / "Your brand isn't a logo" / "Should I start with a workshop?" IEI 2026 IG content pieces.

| Element | Spec |
|---|---|
| **Background** | Cream / warm off-white (`#F4F1EA` or similar — NOT stark white) |
| **Primary type color** | True black (`#0A0A0A`) — heavy weight always |
| **Brand accent** | Yellow (the IEI bulb yellow — looks like `#FFD400` to `#FFE066` range) |
| **Display type** | Heavy-weight modern sans: **Manrope Black / Inter Black** family (or Visby/Söhne if budget allows) — XXL on mobile, takes up space |
| **Body type** | Same family, regular weight — conversational, often lowercase for IG-style content |
| **Yellow accents** | Brush-stroke underlines on key phrases · highlighter blocks behind keywords · yellow boxes behind call-out info |
| **Logo** | Yellow lightbulb mark + "IDEAS EQUAL INCOME" stacked block letters in black |
| **Imagery** | Mix of: (1) vintage B&W stock photography overlaid with warm gradient blobs · (2) real photos of Tab + community · (3) gradient blob backgrounds (orange/pink/blue soft warmth) |
| **Vibe** | "Editorial intellectual with attitude" — smart but not sterile, warm but not soft, confident but not corporate |

### Mode B — IEI Agency Website Vibe (for institutional surfaces: landing About section, press kit, investor decks, partnership outreach)

Source: ieiagency.com per the WebFetch.

| Element | Spec |
|---|---|
| **Background** | Dark navy / charcoal (`#0F1B2D` to `#1A1F2E` range) with white type |
| **Primary type color** | White on dark, charcoal on light sections |
| **Type** | Clean modern sans (likely same family as Mode A — Manrope or Inter) |
| **Spacing** | Generous whitespace, breathing room |
| **Photography** | Warm, approachable, diverse professionals — natural lighting |
| **Vibe** | Sophisticated yet approachable — "we're serious professionals who care" |
| **Logo** | White brandmark on dark backgrounds |

### How the two modes coexist

| Surface | Mode |
|---|---|
| Brand Blueprint landing hero | Mode B (institutional — sells trust) |
| Brand Blueprint "About Tab" section on landing | Mode B with Mode A accents (warm photo + bold pull-quote in yellow underline) |
| Chat intake UI | **Mode A** (cream + yellow + heavy type — warm, conversational) |
| "Lock It In" transition screens | **Mode A** (the lightbulb moment IS the yellow accent) |
| Email templates (welcome, delivery) | **Mode A** (cream + black + yellow accent + Tab-voice copy) |
| Watermark on preview files | Subtle diagonal "PREVIEW · BRAND BLUEPRINT · IEI VENTURES" in low-opacity charcoal |
| Tab's internal Review Dashboard | Mode A (so Tab enjoys using it daily) |
| Investor deck | **Mode B** (dark navy + white + clean) |
| Press kit | **Mode B** |

**Default for FE dev (Buzzy):** when in doubt, use Mode A for customer-touching surfaces and Mode B for credibility/institutional surfaces.

## 🔧 CORRECTIONS to the original doc

### Correction 1 — Tab CAN do quiet luxury (when the brand requires it)

The original doc stated under "Negative Space" that Tab's taste rejects:
- "Pastel-girlboss aesthetic"
- "Minimalism that's actually emptiness"

**Eva Mindful Mondays (executive coach client) contradicts this when read absolutely.** Eva's templates use cream + soft yellow + script monogram + serif quotes + sophisticated executive portrait — that's quiet luxury done well. The pattern reconciles as:

> ✅ **Tab does quiet luxury when the BRAND requires it.** Her own work has volume, but she serves clients whose brands need softness (executive coaches, real estate, certain wellness brands). The rule isn't "no quiet luxury" — it's **"no aesthetic for aesthetic's sake."** If quiet serves the brand, Tab can do quiet.

**Updated negative space rule:** the original list still holds for AESTHETIC for its own sake (pastel-girlboss-as-default, minimalism-as-emptiness, tech-startup-gradient-trust). But each can appear if it's the right tool for the brand.

### Correction 2 — Tab works INTERNATIONAL

Original doc emphasized "Baltimore + community grounding" as a constant. **WeSkate Mongu — Zambia's Second National Skateboarding Championship — proves Tab does international work.** The Baltimore/community-grounding constant should be reframed:

> ✅ **Tab grounds in COMMUNITY (wherever that community is).** Baltimore is her home community; the principle is community-rootedness, not Baltimore-specificity. International work (Zambia) carries the same community-action energy as her local work.

## ⭐ NEW PATTERNS that emerged from the full sweep

### Pattern: Chevron arrows as recurring structural motif
- E.A.S.E. flyer + Infosheet · ZEHBRAS Community CrossFit · several others
- Use as visual anchor + directional cue + textural element
- **Decision rule:** chevron arrows can appear when brand needs movement/momentum/direction without bombast

- VNC Houston "Houston, get ready to go viral!" (script) over "OCTOBER 4-6TH" (bold caps)
- Eva Stories "Conversations with Eva" (script) over "INSPIRED LESSONS" (caps)
- **Decision rule:** script for intimate/personal/invitational moments; bold caps for structural/news/eventful information

### Pattern: Same brand, different palettes per format
- **Eva's brand** = cream + soft yellow for FEED content (Mindful Mondays); cyan + pink for STORIES content (Live broadcasts)
- **D.U.M.I.** = black + red for "Loyalty Collection"; dark forest green + black for "Holiday Sale"; red painted texture for WTG collab
- **Decision rule:** a single brand identity can flex its PALETTE per format/campaign/season without breaking. The brand voice + type discipline stays constant.

### Pattern: Type-only compositions (when the wordplay IS the design)
- Pump-A-Thon @MamaMilkBank "BREAST-FEED THE NEED" — entire frame is two color-blocked words
- Good Business Community "CommUNITY" logo — typographic wordplay carries the whole brand mark
- **Decision rule:** when the wordplay/phrase is strong enough, the design can be JUST type. Don't dilute strong copy with unnecessary imagery.

### Pattern: Multi-host event layouts handled cleanly
- Toni Harris Taylor "Next Level Virtual Networking" — 7 host headshots in circles + main feature portrait + city skyline
- Bridge Builders 3-pillar layouts
- Founders & Feedback 3-up portrait color-block
- **Decision rule:** for multi-person events, use circular crops + grid arrangement; avoid rectangular passport-photo grid (too sterile)

### Pattern: Co-branding cleanly without competition
- Black UX Labs + Uber (logos sit equal at bottom)
- Maryland Neighborhood Exchange + Community Wealth Builders (partner logo as side card)
- Smalltimore Homes + Baltimore Healthy Start (both logos in footer)
- VNC Houston (3 sponsor logos in clean horizontal row)
- **Decision rule:** when partner logos appear, isolate them in a dedicated footer/sidebar zone with consistent sizing — never let them compete with primary brand mark
