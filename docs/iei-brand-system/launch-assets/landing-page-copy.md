## NOTES FOR THE LANDING-PAGE BUILDER

1. **Mobile-first.** Every section must collapse cleanly. Hero video → autoplay muted on mobile, click-to-unmute. Tier cards stack vertically.
2. **Sticky CTA in nav** — visible at all times on scroll.
3. **Three Stripe Payment Links** — wire each tier button directly. Pass `tier=basic|content|full` in URL params for tracking.
4. **IP clickthrough checkbox** — appears at Stripe checkout, NOT on landing. Use the short version from `IP_CLICKTHROUGH.md`.
5. **Tab's video** — record before launch. Loom is fine (free + native sharing) or Riverside for higher quality.
6. **Photo of Tab** — professional headshot in About section. If none ready, use a placeholder + ship without; add later.
7. **No emojis anywhere in copy.** They read as AI-generated.
8. **Hex/typography choices** — match IEI Ventures parent brand (or pick from a Tab-approved palette). Don't let the designer pick "random teal" — get Tab's input.
9. **Trust signals to add post-launch:** real testimonials with photos + names, "as featured in" if applicable, count of Brand Blueprints delivered.
