# Tier Review Checklists — Tab's QC Pass

**Purpose:** What Tab personally reviews on every Blueprint before delivery. Tier-specific so review time matches tier price.

**Time budget per tier:**
| Tier | Review time cap | If exceeds | If exceeds 2× cap |
|---|---|---|---|
| Basic ($997) | 90 min | Refine the 2-3 most critical issues, ship the rest | Refund difference or upgrade customer to Content |
| + Content ($1,997) | 3 hrs | Refine top 5 issues | Flag for second review the next day |
| Full Suite ($3,997) | 5 hrs | Take an extra day | Reschedule delivery + email customer |

**Rule:** Never deliver something below your quality bar. If it's not Tab-quality, regenerate. Better to delay 1 day than ship weak work.

---

## UNIVERSAL CHECKS (every tier)

Before tier-specific review, run these on everything:

- [ ] **No banned vocabulary** in any copy: comprehensive, robust, leverage, delve, transformative, holistic, vibrant, seamless, ecosystem (figurative), tapestry, paradigm, "in today's fast-paced world," "navigate the complexities of"
- [ ] **No emojis** anywhere in playbook, landing, logo, brand.json
- [ ] **No other-brand contamination:** scan for "Aurelian," "Pen2Purpose," "DOL," "AceTV," "FamFit," "Wone," "Banger," "OffScript," "Vent," "Halcyon," "BlkBtrFly" — if any appear, regenerate that section
- [ ] **Customer's name + business spelled correctly everywhere**
- [ ] **Customer's verbatim phrases from intake appear** in brand messaging (their voice > AI's voice)
- [ ] **No "[Recommended — confirm with client]" placeholders** left in delivered copy
- [ ] **Files render correctly** — open the PDF, open the HTML, view the SVG logo
- [ ] **All download links work** in the delivery email
- [ ] **Tier deliverables list matches what's actually in the ZIP** — no missing files, no extras

## TIER 1 — BLUEPRINT BASIC ($997, 90-min review)

**Brand Messaging:**
- [ ] Mission statement = 1 sentence, uses customer's locked one-liner from intake
- [ ] Vision statement = 1 sentence, future-facing, specific
- [ ] 3-5 Core Values, each a short phrase + 1-sentence explanation, in customer's voice
- [ ] 3 Tagline options — each ≤ 7 words, none cliché, at least one a candidate for use TODAY
- [ ] 3 IG Bio options — each ≤ 150 characters, includes 1 emoji-free hook + audience + CTA
- [ ] Brand voice section = "say" list (4-6 phrases) + "don't say" list (4-6 anti-patterns)

**Logo (3 options):**
- [ ] All 3 options on white background AND on dark background
- [ ] All 3 work at 16px (mobile favicon test)
- [ ] All 3 use color count ≤ 3 in primary version
- [ ] At least 1 option has a symbolic element (not just wordmark+dot)
- [ ] Each option has stated rationale (what the shape means, what the color choices represent)
- [ ] Logo aligns with archetype identified in intake (don't ship a playful logo for a "Sage" brand)

**Brand Kit:**
- [ ] Color palette = at least 6 colors (Primary · Secondary · Accent · Background · Text · Muted)
- [ ] Each color has hex + name + 1-sentence usage rule
- [ ] Color contrast pairs pass 4.5:1 minimum for all text/background combos
- [ ] Typography = 3 fonts (display · body · accent), all Google Fonts, each with specimen
- [ ] Font pairing matches archetype (no Cinzel for a Bebas-energy brand, etc.)

**1-Page Website:**
- [ ] Hero matches customer's locked one-liner verbatim
- [ ] CTA matches what they want customers to do (book a call, buy a product, sign up for newsletter — match their stated 6-month goal)
- [ ] Color + font choices match brand kit exactly
- [ ] Mobile responsive (resize browser to phone width, verify)
- [ ] No Lorem Ipsum, no placeholder copy

**GTM Checklist (customized to customer's stated 6-month goal):**
- [ ] References the customer's specific goal from intake Q1 Module 6
- [ ] Maps to The Fourfold Path (Ideation → Impact → Implementation → Income)
- [ ] Lists 8-12 concrete tasks (not vague "build social presence" — specific like "post 3x/week on IG, themed: Mon teach, Wed story, Fri offer")
- [ ] Includes 1 metric per phase to track progress
- [ ] Realistic for a solo founder operating ~10 hrs/week on marketing

**Final pass:**
- [ ] If Tab were the customer, would she be thrilled with this delivery? If "meh" — regenerate the weakest section.

## TIER 2 — BLUEPRINT + CONTENT ($1,997, 3-hr review)

Everything in Tier 1, PLUS:

**Additional Brand Assets:**
- [ ] Social media templates (IG post, IG story, LinkedIn cover) — 3-5 templates, each editable in Canva
- [ ] Business card layout (front + back) — editable
- [ ] Presentation theme (3-5 slide templates) — editable

**First 5 Social Posts (5-Post Framework):**
- [ ] Post 1 — Awareness: Teach Something — has a hook line + 2-4 short paragraphs + clear CTA
- [ ] Post 2 — Awareness: Share Your Story — pulls from customer's narrative (Module 5), NOT generic
- [ ] Post 3 — Trust: Show Proof — uses customer's own results/experience or invites them to add their first one
- [ ] Post 4 — Trust: Connection Post — speaks directly to ideal client's pain (from Module 3)
- [ ] Post 5 — Convert: Present Your Offer — uses locked offer statement from Module 4
- [ ] Each post is in customer's voice (verbatim phrases preserved), not generic AI copy
- [ ] Awareness 50% / Trust 30% / Convert 20% distribution is correct
- [ ] Each post has a suggested hashtag set (5-10 hashtags, mix of branded + audience-tagged)

**Master AI Prompt:**
- [ ] Customer's brand identity, ICA, offer, narrative all populated in the template
- [ ] Prompt is copy-paste ready for ChatGPT, Claude, or any AI tool
- [ ] Includes Tab's voice rules so future content matches
- [ ] Saved as `.txt` file in the delivery ZIP

**Canva Evergreen Templates:**
- [ ] 5-10 templates linked (Tab's existing templates, customer gets edit access via Canva share link)
- [ ] Each template re-skinned with customer's color palette + fonts
- [ ] Templates cover: testimonial post, quote post, before/after, announcement, list/carousel

## TIER 3 — FULL SUITE ($3,997, 5-hr review)

Everything in Tier 1 + Tier 2, PLUS:

**Ideal Client Profile (WS7 deep persona):**
- [ ] Full demographic profile
- [ ] Full psychographic profile (values, fears, aspirations, what keeps them up at night)
- [ ] Behavioral profile (buying triggers, deal-breakers, decision timeline)
- [ ] 12 things they're searching for right now (informs SEO + content topics)
- [ ] Named human archetype (not "Sarah, 35" — specific cultural touchstones)
- [ ] Where they hang out (Dream 100 Lite — specific platforms, podcasts, accounts they follow)

**6-Part Offering Doc (WS8):**
- [ ] All 6 questions answered for their primary offer
- [ ] Value proposition statement (3-5 sentences)
- [ ] Key features + benefits (5-8 each)
- [ ] Distinguishing factors (3-5)
- [ ] Customer pain points (5-7) + how this offering solves each
- [ ] 30-second pitch version (DM-ready)
- [ ] Objection-handling section (3-5 likely objections + responses)

**7-Part Brand Narrative:**
- [ ] All 7 parts written in customer's voice: Origin · Why · Struggle · Turning Point · Who You Serve · How You Help · The Invitation
- [ ] Each part is 100-200 words (long enough to be substantive, short enough to be usable)
- [ ] Includes 5 pulled "hook lines" from the narrative (one-sentence soundbites for socials)
- [ ] LinkedIn-format version (long-post, story arc)
- [ ] Bio-format version (one paragraph, About-page ready)

**6-Week Fourfold Path Roadmap:**
- [ ] Week 1-2 Ideation phase tasks (8-12 tasks)
- [ ] Week 3-4 Impact phase tasks (8-12 tasks)
- [ ] Week 5 Implementation phase tasks (5-8 tasks)
- [ ] Week 5-6 Income phase tasks (5-8 tasks)
- [ ] Each task has: owner suggestion, time estimate, success metric, dependencies
- [ ] Realistic for a solo founder operating ~15-20 hrs/week on launch

**90-Day Content Calendar:**
- [ ] 12 weeks × 3 posts/week = 36 post topics minimum
- [ ] Awareness 50% / Trust 30% / Convert 20% distribution holds across 90 days
- [ ] Includes any holidays / events flagged in intake
- [ ] Each topic has: hook angle, content pillar, CTA, suggested format (carousel, reel, post, story)
- [ ] Themed weeks where applicable (e.g., "Launch Week," "Origin Story Week")

**Priority review + extended Q&A:**
- [ ] 30-min follow-up call scheduled within 14 days of delivery
- [ ] 2 weeks of email/DM Q&A access flagged in delivery email
