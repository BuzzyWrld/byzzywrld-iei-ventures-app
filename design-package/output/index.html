How this was built

Notes for handoff

White-labelable
The primary is a neutral (warm ink), not a brand color. Tenants swap --color-primary / --color-accent via TenantConfig.
Data-bound
Every state comment tagged @bind: project.* matches field names in 03-data-contracts.md.
No AI slop
No sparkles, no gradient blobs, no purple glow. Typography is the visual. Serif moments in intake only — to earn the "brand strategist" feel.
