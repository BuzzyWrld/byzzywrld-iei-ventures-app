Overview

What We're Building

IEI Ventures is a SaaS platform that transforms how businesses establish their brand and acquire clients. A company enters through an intake questionnaire or uploads existing assets — and exits with a complete brand system, a live landing page, and an AI-powered lead generation engine that learns their business.

Feature 02 — AI Engine

Brand Playbook Skill Integration

Skill Handoff

The pre-built Brand Playbook Skill (built by Buzz) is provided to Henrique as the AI core. The dashboard feeds intake data to it via API. No need to build brand logic from scratch.

Skill Outputs

PDF Full brand playbook — 360 DPI, screenshot-rendered via Playwright pipeline
SVG Logo system — primary, icon mark, alt variants via Claude Design
HTML Launch-ready landing page built from brand data, deployable on Vercel/Netlify
JSON Brand data object — colors, fonts, tone, positioning. Powers future skills.

PDF Rendering Pipeline — Confirmed Method

1
HTML Generation Brand playbook rendered as structured HTML with brand colors, fonts, layouts embedded.
2
Playwright Screenshot Chromium renders at 3x device scale. Per-slide screenshot crop. No page.pdf() — eliminates font blur.
3
PIL → 360 DPI PDF PIL assembles screenshots into multi-page PDF at 360 DPI. Pixel-perfect output.
4
Delivery to Dashboard Both HTML + PDF written to outputs. Dashboard surfaces download links in project panel.

Chromium path: /opt/pw-browsers/chromium-1194/chrome-linux/chrome

flags: --font-render-hinting=none --disable-font-subpixel-positioning

Feature 03 — Phase 2

Lead Generation Module

After brand is built, the lead gen agent learns the business model from the brand data already generated — no additional setup. It acts as a persistent radar: monitoring channels, researching prospects, scoring leads, and triggering automated outreach. It also runs internally to acquire clients for the IEI Ventures platform itself.

1
Social Listening
▸ Monitors Reddit, LinkedIn, Twitter/X, forums for intent signals
▸ Keyword triggers matched to client's service offering
▸ Surface companies posting RFPs or seeking rebrands
2
Lead Scoring
▸ Auto-research on each prospect — size, sector, capital signals
▸ Scores viability for client's specific services
▸ Filters out low-capital or mismatched leads automatically
3
Outreach Engine
▸ Personalized email + LinkedIn outreach sequences generated by AI
▸ Sends proposals, sets up discovery calls automatically
▸ Continuously learns what's working — optimizes sequences over time
Internal Use

The same lead gen system runs internally for IEI Ventures — acquiring clients for the platform without paid ads. Cold outreach first → proof of concept → then paid acquisition.

▸ Node.js / Python — backend API layer
▸ Multi-tenant DB — isolated data per client, white label safe
📦

What Henrique Receives

The complete Brand Playbook Claude Code skill is handed off directly — Henrique builds the dashboard and API layer around it. The skill is the AI brain; the dashboard is the interface. Skill outputs (PDF, logo, HTML) are delivered back to the dashboard via API and surfaced in the project panel.

⚠️ PDF Rendering — Critical

Must use screenshot pipeline (Playwright → PIL → PDF). Never page.pdf(). Fonts must load before screenshot. See Slide 05 for full spec.

⚠️ Logo Logic — Critical

AI must evaluate existing assets before generating. Never override a functional logo. Iterate colors/weight first; create new only when no identity exists.

⚠️ No Financial KPIs

Dashboard metrics pull branding/marketing KPIs only. No revenue, P&L, or financial projections — FINRA compliance concern.
