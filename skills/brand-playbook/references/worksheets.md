# Worksheets Reference ‚Äî All 10 Foundational Documents

This file contains the full field structure for each worksheet. Use this when building
the brand foundation in Step 1. Map client input to these fields; infer what's missing.

## WS1: Go-to-Market Checklist (FourFold Path)

This is the master validation checklist. Use it last to confirm all sections are complete.

**Step 1 ‚Äî Ideation (The What)**
- Brand identity defined (WS2 complete)
- Brand messaging complete (WS3 complete)
- Brand style guide created (WS4 complete)
- Visual assets identified (business cards, social templates, covers, website)

**Step 2 ‚Äî Impact (Goal + Intention)**
- SMART Goals created (WS5 complete)
- Market research + competitor analysis done (WS6 complete)
- Customer personas built (WS7 complete)
- Product positioning articulated (WS8 complete)
- Social impact / mission alignment noted

**Step 3 ‚Äî Implementation (The How)**
- Content strategy outlined
- Channel strategy defined (which platforms, why)
- Launch timeline established

**Step 4 ‚Äî Income (Revenue + Growth)**
- Revenue model documented
- Lead gen approach defined
- Referral/partnership strategy noted
- KPIs assigned to each goal

## WS2: Brand Identity (Core Intake)

**Step 1 ‚Äî Define Brand Identity**
- `objectives` ‚Äî Top 3 objectives for brand identity and messaging
- `brand_name` ‚Äî Name of the brand
- `company_background` ‚Äî 3‚Äì4 sentence background on the company/founder
- `tagline` ‚Äî Memorable, impactful tagline or slogan
- `brand_essence` ‚Äî Core personality in 3‚Äì5 words; "if it were a person, it would be..."
- `unique_ability` ‚Äî What makes this brand unique; their superpower / competitive advantage

**Step 2 ‚Äî Mission / Vision / Values**
- `vision` ‚Äî Future state: how will this brand impact the world?
- `mission` ‚Äî Why it exists; the purpose statement
- `core_values` ‚Äî 3‚Äì5 fundamental beliefs and principles (name + 1-sentence explanation each)

**Step 3 ‚Äî Brand Story**
- `brand_story` ‚Äî Narrative: origin, mission, values woven together
- `reason_why` ‚Äî The founder's "why" ‚Äî what prompted this
- `legacy` ‚Äî How they want to be remembered
- `quotables` ‚Äî Common language, phrases, or quotes the brand uses

**Step 4 ‚Äî Brand Persona**
- `personality_traits` ‚Äî 3‚Äì5 adjectives (e.g., bold, empathetic, precise)
- `tone_and_voice` ‚Äî Communication style: formal/informal, authoritative/friendly, etc.
- `interaction_style` ‚Äî How the brand engages: mentor, peer, expert, friend
- `communication_persona` ‚Äî The "character" the brand plays in its messaging

**Step 5 ‚Äî Target Demographics + Value**
- `ideal_clientele` ‚Äî Who the ideal client is (rough profile)
- `value_proposition` ‚Äî Unique value the brand offers; how it solves pain points

**Step 6 ‚Äî Product / Service Offering**
- `offering_1` ‚Äî Name, key features, key benefits
- `offering_2` ‚Äî Name, key features, key benefits
- `offering_3` ‚Äî Name, key features, key benefits

**Step 7 ‚Äî Competitors**
- `competitor_1` ‚Äî Brand name, website, Instagram handle
- `competitor_2` ‚Äî Brand name, website, Instagram handle
- `competitor_3` ‚Äî Brand name, website, Instagram handle

## WS3: Brand Messaging (Derived from WS2)

**Section 1 ‚Äî Brand Identity Copy** (complete last)
- `brand_blurb` ‚Äî 3‚Äì5 sentence intro for strangers; aligned to mission/vision/values
- `instagram_bio` ‚Äî 3 options; punchy, brand-voice-aligned, under 150 characters each

**Section 2 ‚Äî Mission / Vision / Values (Written)**
- `mission_written` ‚Äî Polished mission statement (not a draft ‚Äî final copy)
- `vision_written` ‚Äî Polished vision statement
- `core_values_written` ‚Äî Each value with 2‚Äì3 sentence elaboration in brand voice

**Section 3 ‚Äî Brand Story (Written)**
- `brand_story_written` ‚Äî Compelling narrative version for public-facing use

**Section 4 ‚Äî Brand Persona (Written)**
- `brand_personality_written` ‚Äî Sentences defining: brand personality, brand voice, brand perception, brand language
- For each: brief explanation + example of how it applies

**Section 5 ‚Äî Target Demographics (Written)**
- `target_audience_description` ‚Äî Paragraph describing the audience with demographics + psychographics

**Section 6 ‚Äî Brand Value Statements**
- `brand_positioning` ‚Äî Where the brand sits in the market
- `value_proposition_statement` ‚Äî Single compelling sentence
- `competitive_advantage` ‚Äî What they do that competitors don't/can't
- `unique_selling_proposition` ‚Äî The one thing that makes them the obvious choice

**Section 7 ‚Äî Product/Service Offerings (Formatted)**
Per offering:
- Cost / price range
- Target audience for this specific offering
- Challenge it addresses
- Benefits delivered
- Outcome client experiences
- Call to action

## WS4: Brand Style Guide + Visual Assets

**1 ‚Äî Visual Identity**
- `primary_colors` ‚Äî Hex codes + names (minimum 2)
- `secondary_colors` ‚Äî Hex codes + names
- `accent_color` ‚Äî Hex code + name
- `background_color` ‚Äî Hex code
- `text_color` ‚Äî Hex code
- `do_not_use` ‚Äî Colors to avoid and why
- `heading_font` ‚Äî Name, weight, Google Fonts URL
- `body_font` ‚Äî Name, weight, Google Fonts URL
- `accent_font` ‚Äî (optional) name, weight, use case
- `font_pairing_rules` ‚Äî When each font applies, sizing floors, forbidden combinations

**2 ‚Äî Design Elements**
- `imagery_style` ‚Äî Photography/illustration direction (e.g., "high contrast, desaturated, editorial")
- `icon_style` ‚Äî Line, filled, outlined, custom, etc.
- `pattern_texture` ‚Äî Any repeating elements, their usage rules

**3 ‚Äî Layout + Composition**
- `grid_system` ‚Äî Column count, gutter size, margin rules
- `typographic_hierarchy` ‚Äî H1 ‚Üí H2 ‚Üí H3 ‚Üí Body ‚Üí Caption sizing/weight specs
- `visual_hierarchy_rule` ‚Äî How content is ranked visually (what gets the most attention)

**4 ‚Äî Brand Assets Inventory**
Which of these need to be created (check all that apply):
- [ ] Logo variations (full color, reversed, stacked, horizontal, icon-only)
- [ ] Business cards
- [ ] Social media profile images
- [ ] Instagram feed templates (minimum 9 grid posts)
- [ ] Instagram / Facebook Stories templates
- [ ] LinkedIn cover
- [ ] YouTube banner + thumbnail templates
- [ ] Email signature
- [ ] Presentation deck template
- [ ] General flyer template
- [ ] Website (landing page or full)
- [ ] Brand one-pager / media kit

**5 ‚Äî Logo Usage Rules**
- `minimum_size` ‚Äî Smallest the logo can appear
- `clear_space` ‚Äî Required whitespace around the logo
- `approved_backgrounds` ‚Äî Which backgrounds the logo can appear on
- `logo_donts` ‚Äî Stretching, recoloring, drop shadows, placement restrictions

## WS5: SMART Goals

Build 3‚Äì5 goals. Each goal must satisfy all 5 criteria:

Per goal:
- `goal_statement` ‚Äî Clear, specific objective
- `specific` ‚Äî Who, what, where, when, why (all answered)
- `measurable` ‚Äî The metric (number, percentage, count)
- `achievable` ‚Äî Resources/skills available; obstacles anticipated
- `relevant` ‚Äî How it connects to the brand's mission and vision
- `time_bound` ‚Äî Deadline + interim milestones

Example goal format for the playbook:
> **Goal:** Grow Instagram to 5,000 engaged followers
> **Metric:** Follower count + avg. engagement rate > 4%
> **Timeline:** 6 months from launch
> **Milestone:** 1,000 followers by Month 2

## WS6: Market Research + Competitor Analysis

Run this for 3 competitors (sourced from WS2 Step 7):

Per competitor:
- `name` ‚Äî Brand name
- `product_service_offerings` ‚Äî What they sell
- `usp` ‚Äî Their unique selling proposition (as they present it)
- `target_audience` ‚Äî Who they go after
- `strengths` ‚Äî What they do well
- `weaknesses` ‚Äî Where they fall short
- `opportunities` ‚Äî Gaps the client brand can exploit against them
- `threats` ‚Äî What this competitor does that could hurt the client

After all three, synthesize:
- `market_position_summary` ‚Äî Where the client brand fits in the landscape
- `key_differentiator` ‚Äî The one thing the client does that none of the three do

## WS7: Target Audience + Customer Persona

**Step 1 ‚Äî Demographics**
- Age range
- Gender
- Location (global / regional / city-level)
- Income level
- Education level
- Occupation / industry
- Marital status
- Family size

**Step 2 ‚Äî Psychographics**
- Lifestyle
- Interests
- Values
- Attitudes toward the category
- Hobbies
- Personality traits
- Challenges / pain points (minimum 3)
- Behavioral characteristics
- Buying behavior (how they research, decide, purchase)

**Step 3 ‚Äî Primary Audience Segment** (the ICA ‚Äî Ideal Client Avatar)
- Give the persona a name and age
- Write a 3‚Äì4 sentence narrative describing their day, their problem, and why this brand is the answer
- List their top 3 buying triggers
- List their top 3 objections

**Step 4 ‚Äî Marketing Channel Mapping**
- Where is this persona spending time online?
- Which channels are priority (ranked 1‚Äì5)?
- What type of content do they engage with?

## WS8: Product Positioning

Per offering (up to 3):
- `offering_name`
- `average_market_cost` ‚Äî What competitors charge for this
- `key_features` ‚Äî What it includes / does
- `key_benefits` ‚Äî What the client gets / feels
- `whats_included` ‚Äî Deliverables, scope, timeline
- `distinguishing_factors` ‚Äî What makes this version better/different
- `ideal_customer` ‚Äî Which persona this offering is for
- `customer_pain_points` ‚Äî The exact problem this solves
- `how_it_solves` ‚Äî Mechanism of the solution
- `value_proposition_statement` ‚Äî Single sentence: "We help [who] do [what] so they can [outcome]"

## WS9: Niche Markets

Identify 3‚Äì5 niche segments where the ICA is concentrated.

Per niche:
- `niche_name` ‚Äî E.g., "Women-owned service businesses in NYC"
- `trends` ‚Äî What's changing in this niche right now
- `customer_needs` ‚Äî What they're actively looking for
- `price_analysis` ‚Äî What they're used to paying; price sensitivity
- `distribution_channels` ‚Äî How they find vendors (referral, Instagram, LinkedIn, events, etc.)
- `key_insights` ‚Äî The 1‚Äì2 things that make this niche worth targeting
- `action_plan` ‚Äî How to penetrate this niche (2‚Äì3 tactics)

## WS10: Dream 100

**Core question:** Does the ICA desire more ‚Äî wealth, health, or relationships?
(This determines the emotional hook for all marketing)

**12 ICA Pain Points + Obsessions**
Using the primary audience segment from WS7, list 12 things this person:
- Is thinking about
- Is talking about
- Is frustrated by
- Is searching for answers on

Split across two segments if the brand serves both SMBs and individuals.

**Where Are They Hiding?**
List the specific locations ‚Äî not just "Instagram" but:
- Which accounts are they following?
- Which hashtags?
- Which Facebook groups?
- Which newsletters?
- Which events / conferences?
- Which podcasts?
- Which influencers / thought leaders?

**Dream 100 List**
100 specific people, brands, channels, and communities where the ICA is already gathered.
(In the playbook, show the top 20 ‚Äî the full list is a working document)
