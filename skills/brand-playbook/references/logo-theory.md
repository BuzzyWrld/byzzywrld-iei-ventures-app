# Logo Design Theory Reference — Brand Playbook

This reference is loaded during **Step 2: Design the Brand Kit → Logo System** section.
It governs how the LLM evaluates, documents, and specifies the rules for any logo — whether
provided by the client or inferred/described conceptually. Apply every applicable rule when
building the Logo System section of the playbook.

---

## How to Use This Reference

When documenting or designing the logo system:

1. Identify the **logo type** (Section 1) — determines which rules apply
2. Apply **form and composition principles** (Section 2) — the underlying structure
3. Apply **color-in-logo rules** (Section 3) — separate from brand palette; logos have stricter rules
4. Apply **sizing and scaling rules** (Section 4) — determines minimum size, safe zones, print vs. digital
5. Document **forbidden treatments** (Section 5) — non-negotiable no's
6. Build the **logo variants system** (Section 6) — all logos need multiple versions

---

## Section 1: Logo Typology

Identify which type the client's logo falls into. This affects everything downstream.

| Type | Description | Examples |
|---|---|---|
| **Wordmark** | Brand name as styled typography, no icon | Google, FedEx, Coca-Cola |
| **Lettermark / Monogram** | Initials or abbreviation, stylized | IBM, CNN, LV (Louis Vuitton) |
| **Pictorial Mark** | Standalone icon/symbol, no text | Apple (bite), Twitter bird, Nike swoosh |
| **Abstract Mark** | Non-representational geometric form | Pepsi circle, Adidas stripes, Chase octagon |
| **Combination Mark** | Icon + wordmark, used together or separately | Starbucks, Mastercard, Amazon |
| **Emblem / Seal** | Text integrated into a badge, crest, or seal shape | Harley-Davidson, Harvard, government seals |
| **Dynamic / Responsive Mark** | Logo that changes form based on context | MIT Media Lab, Google Doodles |

**Most client brands use Combination Marks.** Document both elements separately — the icon and
the wordmark — because they will be used independently across applications.

## Section 2: Form and Composition Principles

These are the structural rules that make a logo visually competent and scalable.

### The Simplicity Principle
A logo must be recognizable at 16px (favicon) and legible at 16 feet (billboard). The more complex
the form, the more it fails at small sizes and distance. The test: can you draw it from memory
in under 5 seconds? If not, it's too complex.

- **Reduce to essence**: Remove any element that isn't doing functional work
- **One focal point**: The eye can't rest on two places at once — choose the anchor
- **No more than 3 visual elements** in a single logo lockup

### Geometric Anchoring
Strong logos are built on geometric primitives — circles, triangles, squares, diagonals.
The viewer's brain pattern-matches to geometry faster than any other form.

- **Circle**: Unity, wholeness, community, infinite (no beginning/end) — used in seals, wellness, community brands
- **Square / Rectangle**: Stability, reliability, order, institutional authority — used in finance, legal, corporate
- **Triangle**: Direction, aspiration, forward movement, hierarchy — used in tech, growth, energy
- **Diagonal lines**: Dynamism, speed, progress — used in motion/action-oriented brands
- **Organic/flowing forms**: Approachability, creativity, naturalness — used in lifestyle, wellness, food

### Negative Space
The most powerful logos exploit negative space — the shape formed by the absence of marks.
Examples: FedEx arrow (hidden between E and X), WWF panda (white space defines the shape),
Amazon smile (also points from A to Z). Always look at what the space between elements forms.

### Visual Weight and Balance
- **Symmetry**: Signals formal authority, trust, stability — appropriate for institutional brands
- **Asymmetry**: Signals dynamism, modernity, creativity — appropriate for innovative/disruptive brands
- **Centered composition**: Static, formal, authoritative — seals, crests, institutional marks
- **Off-center composition**: Active, modern, directional — startup, lifestyle, growth brands

**Weight balance rule**: Every logo must have a visual center of gravity. If it were a physical
object, it should not tip over. Test by blurring the logo — the blur cloud should be roughly centered.

### The Golden Ratio and Proportion
Many enduring logos use the golden ratio (1:1.618) for proportional relationships — between
icon and wordmark width, between letterform heights, between thick and thin stroke weights.
This is not required but is the underlying reason many logos feel "right" without the viewer
knowing why.

### Icon-to-Wordmark Ratio (for combination marks)
- **Standard horizontal lockup**: Icon occupies 20–30% of total width, wordmark 70–80%
- **Stacked lockup**: Icon above, wordmark below — icon no larger than 2× the cap height of the wordmark
- **Never let the icon dominate the wordmark at first use** — the icon earns the right to stand alone over time

## Section 3: Color Application in Logos

Logo color operates under stricter rules than the brand palette. A logo must function in contexts
where brand palette colors cannot be guaranteed — embroidery, engraving, newspaper print, embossing,
single-color stamp, white paper, dark backgrounds, signage. Plan for all of them.

### The Master Rule
**Every logo must be designed first in black on white.** If it doesn't work in black on white,
it doesn't work. Color is applied after form is confirmed. A logo that only works in color is
not a real logo.

### Color Count Rules for Logos
- **Maximum 3 colors** in the primary color logo version
- **1–2 colors** is ideal for versatility and production cost
- Every color version of the logo must have a 1-color (black), 1-color (white/reversed), and
  1-color (PMS/brand primary) version

### Color Meaning Within the Logo
The primary color of a logo carries a heavier psychological burden than the same color in the
broader palette — because the logo is the brand's face. Rules:

- The logo's primary color must be the most psychologically loaded color in the palette for the brand's core promise
- The accent color in the logo (if any) should be the tension or energy color — the one that signals what makes the brand different
- Never put the muted/neutral color as the primary logo color — it reads as low-confidence

### Multi-Color Logo Rules
When a logo uses 2–3 colors:
- **Dominance rule**: One color must occupy more than 50% of the colored area — never a 50/50 split
- **Contrast rule**: Adjacent colors must have a minimum 3:1 contrast ratio
- **Color relationships**: Use split-complementary or analogous pairs — never direct opposites without a neutral separator (the clash will make the logo vibrate visually)

### Gradient Use in Logos
Gradients in logos are a liability:
- Cannot be embossed, engraved, or screen-printed in 1 color
- Often fails in fax, photocopy, or low-res digital reproduction
- If a gradient logo is provided by the client, the logo system MUST include a flat-color version as the primary production-safe mark
- **Approved gradient use case**: Digital-only brands (apps, SaaS products) where the logo will never appear in print

### Logo on Backgrounds — Required Test
Every logo must be documented on at minimum these 5 backgrounds:
1. Pure white
2. Primary brand color (dark or light field)
3. Secondary brand color
4. Black
5. Photography/real-world context background (mid-value image)

If the logo is illegible or disappears on any of these, a variant must be created for it.

## Section 4: Sizing and Scaling Rules

### The Scaling Hierarchy
Every logo must be designed as a **responsive system** — not a single file. The logo changes
in complexity as it shrinks. This is called responsive logo design.

| Size Tier | Context | Logo Complexity |
|---|---|---|
| **Full lockup** | Hero sections, print collateral, presentations above 200px wide | Full combination mark: icon + wordmark + tagline if applicable |
| **Standard** | Most digital applications, email headers, social profile banners | Icon + wordmark, no tagline |
| **Compact** | App nav bars, mobile headers, small print applications | Icon + wordmark horizontal, no descriptor |
| **Icon / App mark** | App icons, favicons, social profile photos, embroidery | Icon / monogram only — no wordmark |
| **Micro** | Favicon (16px), watch face, emboss stamp | Simplified icon with maximum 2–3 elements — must read as a shape |

**The critical failure**: Using the full lockup at micro sizes. A 16px favicon of the full combination
mark is unreadable. Every brand needs a simplified icon mark that is distinct, not just the logo shrunk.

### Minimum Size Rules
- **Digital minimum (full lockup)**: 120px wide
- **Print minimum (full lockup)**: 1.25 inches wide
- **Digital minimum (icon/app mark)**: 32px
- **Print minimum (icon/app mark)**: 0.5 inches

Below these sizes, the logo cannot render its detail correctly. Document these minimums explicitly
in the Logo System section of the playbook.

### Clear Space (Exclusion Zone)
Every logo must have a defined exclusion zone — the minimum blank space that must surround
the logo at all times. No text, imagery, or other graphic element may enter this zone.

**Standard rule**: Clear space = cap-height of the first letter of the wordmark on all four sides.
This is expressed in the brand kit as "equal to the height of [letter]" — not as a fixed pixel value —
because it scales with the logo.

For round/emblem logos: Clear space = radius of the outermost circle on all sides.

### File Format Requirements
Specify which format for which use:
- **SVG**: All digital applications — scalable to any size without quality loss
- **EPS / AI**: Print production — master vector files
- **PNG (transparent background)**: Digital applications where SVG is not supported (social, email)
- **PNG (white background)**: Never use as a master file — use only for presentations/mockups
- **JPG**: Never acceptable for logo files — lossy compression degrades edges
- **PDF**: Single-use print production delivery

## Section 5: Forbidden Logo Treatments

These are the non-negotiable rules. Add these to the Logo Rules section of every playbook.

### Universal Forbidden Treatments
- **Never stretch or distort proportions** — maintain aspect ratio at all times
- **Never rotate the logo** (unless the logo is specifically designed with a rotated element)
- **Never add drop shadows, glows, or outer strokes** to the logo
- **Never place the logo on a busy background** without an approved lock-up treatment (contained background plate)
- **Never rearrange elements** — icon position relative to wordmark is fixed
- **Never apply unauthorized color changes** — only approved color versions exist
- **Never use low-resolution versions** — minimum print: 300dpi, digital: SVG or high-res PNG
- **Never add text to the logo** that isn't part of the original system
- **Never use the wordmark as body text** — it is a logo, not a typeface
- **Never crop the logo** to show only part of it

### Geometry-Specific Rules (for logos with geometric elements)
- **Angles must be preserved exactly**: A circuit logo using 90°/45° angles cannot be modified to use 30° or 60° angles
- **Stroke weights are fixed**: Never increase or decrease the stroke weight of a geometric logo element — it changes the visual weight balance
- **Grid alignment**: Any logo with internal geometric structure must stay on its original grid — do not eyeball spacing modifications

### Emblem/Seal Specific Rules
- **Interior elements have a fixed minimum size** relative to the overall emblem — never scale the emblem without preserving internal proportions
- **Circular text must follow the original kerning and tracking** — never re-typeset seal text independently

## Section 6: Logo Variant System (required for every brand)

Every brand's logo system must include these variants. Document each with its approved use case.

### Variant 1: Primary Color Lockup
Full combination mark in the primary brand colors on the standard brand background.
This is the "official" version — used in most formal communications.

### Variant 2: Dark Field Version
The logo adjusted for use on dark/primary-colored backgrounds. Usually: wordmark/icon in white
or ivory, with accent element in the gold/accent color.
**Required for**: presentations, merchandise on dark apparel, website hero sections.

### Variant 3: Reversed / White Version
All-white version on transparent background.
**Required for**: dark photography overlays, social media over varied images, embossing on dark materials.

### Variant 4: All-Black Version (Production Mark)
All-black, single color, on transparent background.
**Required for**: fax, photocopied documents, newspaper ads, screen printing, embroidery, legal filings.

### Variant 5: Monogram / App Icon
Simplified mark — typically the icon or initials only, without wordmark.
**Required for**: favicons (16×16, 32×32), app store icons (1024×1024), social profile photos,
emboss/deboss stamps, small watermarks, email signature monogram.

### Variant 6: Horizontal Lockup (if default is stacked)
If the primary lockup is stacked (icon above wordmark), a horizontal version is required for
narrow contexts: email headers, website nav bars, letterhead footers.

### Variant 7: Sub-brand / Division Mark (if applicable)
For companies with multiple divisions or products under one parent brand: a modified version
that adds the division name while maintaining the core mark structure.

## Section 7: Logo Documentation in the Playbook

When rendering the Logo System section of the playbook, each variant gets a documentation card:

```
VARIANT NAME: [Primary Color / Dark Field / Reversed / All-Black / Monogram]
APPROVED BACKGROUNDS: [List of approved backgrounds]
MINIMUM SIZE (digital): [Xpx]
MINIMUM SIZE (print): [X inches]
CLEAR SPACE: [Equal to cap-height of [letter] on all sides]
FILE FORMAT: [SVG for digital / EPS for print / PNG for web]
USE CASE: [Where this specific variant is used]
FORBIDDEN ON: [Backgrounds/contexts where this variant must not appear]
```

Render each variant as a visual mock — show the logo on its approved background, not floating
in empty white space. This is how brand standards are communicated to vendors, designers, and
internal teams who will execute against the brand.
