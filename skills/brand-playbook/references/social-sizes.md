# Social Media Image Sizes Reference
Source: Sprout Social — Last Updated January 31, 2025

Use this when building the Brand Assets Inventory section of the playbook
and when creating any social media templates for a client.

---

## Instagram
| Asset | Dimensions | Aspect Ratio |
|-------|-----------|--------------|
| Profile Picture | 110 × 110 | 1:1 |
| Feed Post (Square) | 1080 × 1080 | 1:1 |
| Feed Post (Portrait) | 1080 × 1350 | 4:5 |
| Stories / Reels | 1080 × 1920 | 9:16 |

**Grid template note:** When building Instagram grid templates, use 1080×1080 (square)
for the standard feed post. Design in sets of 3 or 9 for grid coherence.

---

## Priority Asset Set for New Brand Launches

For a standard brand launch, build these first (in this order):
1. Instagram Feed Post template — 1080 × 1080 (9-grid set)
2. Instagram Stories template — 1080 × 1920
3. Facebook/LinkedIn Cover — 1128 × 191
4. YouTube Banner — 2560 × 1440
5. Profile Images — 400 × 400 (works across most platforms)
6. YouTube Thumbnail template — 1280 × 720
